## Working thesis

Decision agent 是一种通过 multi-agent workers team 协同完成任务的决策架构，可作为独立组件部署。Openclaw 则是一种 7×24 持续运行的个人助理形态，与 decision agent 关系互补：openclaw 扮演经理角色，decision agent 充当底层员工 team。二者结合后，长期助理获得任务执行能力，decision agent 获得持续上下文与调度入口。本课题的关注点是 decision agent 的技术路径与未来方向，需结合最新业界进展持续追踪。该论点可被证伪——若发现主流 decision agent 必须依附于长期助理上下文才能闭环，或两者结合并不带来明显增益，则需要修正。
